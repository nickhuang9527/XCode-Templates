## Installation
===
```bash=
cd ~/Library/Developer/Xcode
mkdir Templates
mkdir ${Template_Name}
cd ${Template_Name}
mv xxxx.xctemplate .
```
[Template_Name]
> MVC Template, MVVM Template, VIPER Templates ...
